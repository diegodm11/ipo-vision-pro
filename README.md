# Trabajo IPO — Dispositivo de interacción (Apple Vision Pro)

Este repositorio contiene una **página web estática** (HTML + CSS) que analiza Apple Vision Pro como dispositivo reciente y relevante para la Interacción Persona‑Ordenador.

## Estructura
- `index.html`: contenido principal en una única página (con menú superior).
- `styles.css`: hoja de estilos única que controla la presentación.

## Cómo publicar en GitHub Pages
1. Crea un repositorio nuevo en GitHub (p. ej., `ipo-vision-pro`).
2. Sube `index.html` y `styles.css` a la rama `main`.
3. Ve a **Settings → Pages** y elige **Source: Deploy from a branch** → `main` / `/ (root)`.
4. Guarda. GitHub publicará en una URL del tipo `https://TU_USUARIO.github.io/ipo-vision-pro/`.
5. Copia esa URL en `entrega_url.txt` y súbela a Moodle.
